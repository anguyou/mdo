Use Less.js --- I've been using Less.js in just about every project I can lately and I freaking love it. I don't go hog wild about it though; I'd rather preserve the readability and structure of regular ol' CSS. That said, I do want to make more complex things easier. But wait, why use it? Well, for a few reasons really, but most importantly to add variables for repeated values like colors and to simplify the usage of CSS3 values that have vendor prefixes.
