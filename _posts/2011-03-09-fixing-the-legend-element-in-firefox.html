Firefox has this weird janky implementation of the legend HTML element. It doesn't handle margins correctly and as a result it can cause some headaches. But there is hope!

Here's the setup: using a legend element with a margin-left within a fieldset with a border-top doesn't work.

  <fieldset>
    <legend>Fieldset legend</legend>
    <div class="clearfix">
      <label for="input">Input label</label>
      <input type="text" id="input" placeholder="Type in me!" />
    </div>
  </fieldset>
</form>
